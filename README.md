# 📄 Description:

### What is My Repos? 🤔

- A project made with React that allows you to save your favorite repositories on Github. 🚀

# 🚀 Tecnologies:

This project was developed with the following technologies:

### React 📝

### react-router-dom 📝

### JavaScript 📝

### react-icons 📝

### axios 📝

### Github API 📝

### Styled Components 💅

How to run:

```bash

# Clone Repository

$ git clone

# Go to folder

$ cd my-repos

# Install Dependencies

$ npm install

# Run Aplication

$ npm start

```

Go to http://localhost:3000/ to see the result.
